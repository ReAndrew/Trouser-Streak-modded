## Features:
All features from default TS but added a new one and modified some more!
- [ForceOp Spawner] Uses a bug of creating commands inside a entity 
- [ForceOp Command Block] Uses a spawn egg OR a command block with auto op player. And add a feature for spawning command block up like on 255 Y for ForceOP your nick
- (modded) [ForceOPSign] Now allowing paste text inside!
- [AutoCommandBlock] Puts command blocks with execution code
- [AutoCorrupt] Trying to corrupt the world using some bad commands
- [OPDetector] Trying to detect people with OP using chat logs
- [MemoryLeak] Spawn a lot of entity with garmage data (NBT, crash shulker and other)
- (Porting from addons / LiquidBounce) [Server crasher] Crashes server using some code
- [PowerNBT] Can inspect and modify NBT data from item in hand!
- (modded) [Server Info] Add ViaFabricPlus support for version
- [MeteorCFG] Generates or imports an a Full CFG with binds and other to file. Allows to control all commands in Meteor and disable some!
